# Brazilian E-Commerce SQL Analysis

## Overview
SQL analysis of 100,000+ real orders from the Olist Brazilian e-commerce dataset sourced from Kaggle.

## Questions Answered
1. How many orders were placed in total?
2. How many orders were delivered successfully?
3. Which month had the most orders?
4. What is the total revenue generated?
5. What are the top 5 product categories by revenue?
6. What is the average payment value per order?
7. How many orders had bad reviews?
8. Which sellers have the most orders?

## Tools Used
- DB Browser for SQLite
- SQLite

## Files
- `ecommerce_analysis.sql` — all queries with comments
- `ecommerce_analysis.pdf` — full analysis with results and insights
